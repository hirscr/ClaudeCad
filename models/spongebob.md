# SpongeBob SquarePants (10cm tall)

Create SpongeBob as a multi-colored 3D model using Compound (not BuildPart).

## Coordinate System
- Z = up/down, Y = forward/back, X = left/right
- Face looks toward +Y
- Units: millimeters

---

## 1) Body (yellow sponge rectangle)
- Box: 45mm wide × 25mm deep × 50mm tall
- Center at (0, 0, 55)
- Color: yellow

---
