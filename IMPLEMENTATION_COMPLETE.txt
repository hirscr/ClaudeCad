## Key Implementation Details

### Flow with Images:
1. Renderer calls: ipc.invoke('send-chat-message', { message, currentCode, history, clickInfo, imagePaths: [{number: 1, path: "/tmp/..."}] })
2. Main extracts imagePaths parameter
3. buildPrompt() inserts image references into prompt:
   ```
   # Reference Images
   
   The user has provided the following images:
   - Image 1: /tmp/claudecad-images/img_001.png
   ```
4. Main determines tempImageDir = TEMP_IMAGE_DIR if images exist
5. sendPrompt(prompt, tempImageDir) adds --add-dir flag:
   ```
   spawn('claude', ['--print', '--add-dir', '/tmp/claudecad-images'])
   ```
6. Claude CLI has access to read image files

### Flow without Images:
- imagePaths = undefined or null or []
- tempImageDir = null
- No --add-dir flag added
- No image section in prompt
- Works exactly as before
